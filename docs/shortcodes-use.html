LEANWI-Book-A-Room

Setting up Pages Using Shortcodes

Venue Pages (Venue Example Page)

Venues are configured in the plugin settings, but each venue needs a page with the corresponding shortcode:

Example: [venue_details venue_id="1"] (for venue ID 1).

This is a required page for each individual venue/room.

Check Availability Page (Room Availability Example Page)

This page allows users and staff to check venue availability for a selected day using the [venue_grid] shortcode.

Staff can also search for bookings by name or organization using [staff_name_search].

This is a required page for the functionality of the LEANWI-Book-A-Room system.

Recurring Bookings Page (Recurring Bookings Example Page)

Staff can create recurring bookings by adding the shortcode [staff_recurring_bookings] to a page.

This is only required if you wish to add recurring bookings to a venue such as Storytime or regular meetings.

Payments and Feedback Monitoring Page (Optional) (Payments and Feedback Example Page)

To track payments and request feedback, add the shortcode [staff_payment_search] to a page.

This is only required if you have costs associated with your rooms or wish to ask for feedback.
