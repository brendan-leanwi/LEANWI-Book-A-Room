LEANWI-Book-A-Room

Adding a mail client

It may be necessary to set up a mail client to allow you to send emails from a validated email address which will increase the chance that emails sent from LEANWI-Book-A-Room arrive in the users inbox.

If you do not already have a preferred mail client then the WP Mail SMTP plugin seems like a good option.

But please feel free to investigate other options for adding a mail client to your WordPress site.
